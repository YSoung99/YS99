1. 원문과 번역문을 입력하고 추가 버튼을 누르세요.

2. 번역 시 사전에 있는 단어나 문구는 자동으로 대체됩니다.

3. 항목 관리 버튼으로 개별 항목을 삭제할 수 있습니다.

사용 방법

  1. 설정 탭에서 Gemini API 키를 입력하고 저장합니다.
  2. 번역 탭에서 출발어와 도착어를 선택합니다.
  3. 번역할 텍스트를 입력하면 자동으로 번역이 시작됩니다.
  4. 번역 결과를 복사하려면 "복사하기" 버튼을 클릭하세요.
